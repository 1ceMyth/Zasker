Zasker is a full-stack web application that connects companies with users who solve real-world problems for monetary rewards. It simulates a crowdsourcing and freelancing environment where companies post challenges and users earn by solving them.

## Features

### User
- Sign up / Login
- Create profile (name, skills, bio)
- Browse problems by category and difficulty
- View reward amounts
- Mark problems as "In Progress"
- Submit solutions
- View attempted problems and total earnings

### Company
- Company signup / login
- Post new problems
- Set difficulty and reward
- View submitted solutions
- Accept or reject submissions

### Pages
- Home page
- Authentication pages
- Problem listing
- Problem detail
- User dashboard
- Company dashboard

## Tech Goals
- Clean, modern UI
- Authentication system
- Role-based dashboards
- Simple database or in-memory storage
- Fully runnable demo project
- No real payment integration

## Purpose
This project demonstrates:
- Full-stack architecture
- Role-based authentication
- Real-world workflow simulation
- Problem-solution lifecycle
- Dashboard-based UX

Zasker is designed as an academic and demo project focusing on functionality and clarity.
